# **App Name**: KapdaKart

## Core Features:

- Product Catalog: Display clothing products with images, descriptions, and prices.
- Product Detail Page: Detailed view for each product, including zoomable images and size options.
- Order Form: Collect customer information (name, address, contact details) after they click 'Order'.
- WhatsApp Order Submission: Automatically send the collected customer details and order information to +91 93323 07996 via WhatsApp.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) for a sophisticated and trustworthy feel.
- Background color: Very light lavender (#F0F2FA). It provides a soft contrast that keeps the deep indigo prominent.
- Accent color: Warm Orange (#FF9800). Use this color to draw attention to vital parts like order button.
- Body and headline font: 'PT Sans' for a balance of modern and warmth. Easy to read on all devices
- Use simple, line-based icons for categories and UI elements.
- Clean, grid-based layout with ample spacing for readability.
- Subtle transitions on hover and form submission to enhance user experience.
